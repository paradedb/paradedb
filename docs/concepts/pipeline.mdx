---
title: "Pipeline"
---

`Pipeline` defines the entire pipeline, from source to sink.

## Definition

```bash
pipeline = Pipeline(
    source=source,
    transform=transform,
    embedding=model,
    sink=sink,
    target=target,
)
```

**Parameters**

<ParamField body="source" type="Source" required>
  A `Source` object
</ParamField>
<ParamField body="transform" type="Transform" required>
  A `Transform` object
</ParamField>
<ParamField body="embedding" type="Embedding" required>
  A `Embedding` object
</ParamField>
<ParamField body="sink" type="Sink" required>
  A `Sink` object
</ParamField>
<ParamField body="target" type="Target" required>
  A `Target` object
</ParamField>

## Methods

### pipe

```bash
pipeline.pipe(
    embeddings = [[0.1, 0.2, 0.3], [0.2, 0.4, 0.6]],
    ids = ["id1", "id2"],
    # Optional
    metadata = [{"key1": "value1"}, {"key2": "value2"}]
)
```

**Parameters**

<ParamField body="ids" type="List[Union[str, int]]" required>
  An array of IDs corresponding to the embeddings or documents array
</ParamField>
<ParamField body="embeddings" type="List[List[float]]">
  An array of embeddings
</ParamField>
<ParamField body="documents" type="List[str]">
  An array of documents. Retake will automatically embed these documents using
  the embedding model provided in the `Embedding` object. Either `embeddings` or
  `documents` arguments must be provided, but not both
</ParamField>
<ParamField body="verbose" type="boolean" default="True">
  Will print a progress bar if set to `True`
</ParamField>

**Parameters**

<ParamField body="verbose" type="boolean" default="True">
  Will print a progress bar if set to `True`
</ParamField>
