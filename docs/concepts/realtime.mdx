---
title: "RealtimeServer"
---

`RealtimeServer` specifies the connection details of the real-time server. In order to connect to
a real-time server, you must first deploy one. See the [Deployment](/deployment/local) section.

## Definition

```bash
realtime_server = RealtimeServer(
  host="localhost",
  broker_port=9094,
  schema_registry_port=8081
)
```

**Parameters**

<ParamField body="host" type="string" required>
  Host address of the real-time server
</ParamField>
<ParamField body="broker_port" type="int" required>
  Kafka broker port. See `docker-compose.yml` for the default broker port.
</ParamField>
<ParamField body="schema_registry_port" type="int" required>
  Kafka schema registry port. See `docker-compose.yml` for the default schema
  registry port.
</ParamField>
