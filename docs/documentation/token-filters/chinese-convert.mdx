The Chinese convert filter converts text between Traditional and Simplified Chinese characters, enabling unified search across both character sets. It is supported for all tokenizers besides the [literal](/documentation/tokenizers/available-tokenizers/literal) tokenizer.

To enable Chinese conversion, append `chinese_convert=<mode>` to the tokenizer's arguments.

## Conversion Modes

The following conversion modes are available:

- **`t2s`** - Traditional to Simplified Chinese
- **`tw2s`** - Traditional Taiwan to Simplified Chinese
- **`tw2sp`** - Traditional Taiwan to Simplified Chinese (with idioms)
- **`s2t`** - Simplified to Traditional Chinese
- **`s2tw`** - Simplified to Traditional Taiwan Chinese
- **`s2twp`** - Simplified to Traditional Taiwan Chinese (with idioms)

## Example

### Basic Conversion

To demonstrate this token filter, let's compare the output of the following statements:

```sql
SELECT
  '繁體中文測試'::pdb.jieba::text[],
  '繁體中文測試'::pdb.jieba('chinese_convert=t2s')::text[];
```

```ini Expected Response
        text         |        text
---------------------+---------------------
 {繁體,中文,測試}     | {繁体,中文,繁体中文,测试}
(1 row)
```

### Simplified to Traditional

```sql
SELECT
  '简体中文测试'::pdb.jieba::text[],
  '简体中文测试'::pdb.jieba('chinese_convert=s2t')::text[];
```

```ini Expected Response
        text         |        text
---------------------+---------------------
 {简体,中文,简体中文,测试}     | {簡體,中文,測試}
(1 row)
```

### Unified Search

By converting all text to a single character set during indexing, you can search with either Traditional or Simplified Chinese and match both:

```sql
CREATE TABLE articles (
    id SERIAL PRIMARY KEY,
    title TEXT,
    content TEXT
);

INSERT INTO articles (title, content) VALUES
    ('繁體標題', '這是繁體中文的內容'),
    ('简体标题', '这是简体中文的内容'),
    ('運動鞋', '適合跑步和運動的鞋子'),
    ('运动鞋', '适合跑步和运动的鞋子');

-- Create index with T2S conversion
CREATE INDEX articles_idx ON articles
USING bm25 (id, (title::pdb.jieba('chinese_convert=t2s')), (content::pdb.jieba('chinese_convert=t2s')))
WITH (key_field='id');

SELECT id, title FROM articles WHERE title ||| '标题' ORDER BY id;
```

This query returns:

```ini Expected Response
 id |  title
----+----------
  1 | 繁體標題
  2 | 简体标题
(2 rows)
```

### Combining with Other Filters

Chinese conversion can be combined with other token filters:

```sql
SELECT '繁體中文測試'::pdb.jieba('chinese_convert=t2s', 'remove_short=2', 'remove_long=10')::text[];
```
