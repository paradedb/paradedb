---
title: Literal Normalized (Deprecated Alias)
description: Deprecated alias for the normalized tokenizer
canonical: https://docs.paradedb.com/documentation/tokenizers/available-tokenizers/literal-normalized
---

`pdb.literal_normalized` is a compatibility alias for `pdb.normalized`.

For new indexes and examples, use [normalized](/documentation/tokenizers/available-tokenizers/normalized).

```sql
-- preferred
SELECT 'Tokenize me!'::pdb.normalized::text[];

-- compatibility alias
SELECT 'Tokenize me!'::pdb.literal_normalized::text[];
```
